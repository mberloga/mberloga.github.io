Что такое GitHub

Веб-сервис GitHub востребован для хостинга IT-проектов и совместной разработки кода. Разработчики системы называют её «социальной сетью» для программистов. Здесь они объединяют репозитории, комментируют примеры «чужого» кода и используют платформу в качестве облачного хранилища с возможностью быстрой передачи заказчику.

Альтернативы: GitLab, BitBucket, CodeRoad, GitFlic (РФ) и другие.

Создание Репозитория

Важно отметить, что сервис англоязычный, и пользоваться им без знания языка получится только при использовании браузеров, где есть встроенные функции по переводу страниц. В любом случае работа начинается с создания собственного Репозитория – доступны публичные и приватные.

Последовательность действий:

  1. Нажать на кнопку «Start a project».
  2. Ввести название и описание Репозитория.
  3. Поставить галочку на «Initialize this repository with a README».
  4. Выбрать нужный тип лицензии и нажать на кнопку «Create project».

Сайт на GitHub Pages создается из репозитория, который должен быть публичным (то есть все ресурсы проекта будут видны другим пользователям) и иметь специальное имя. Такой репозиторий может быть всего один на одном аккаунте GitHub.

Какой тип файлов можно загружать на GitHub?

GitHub не ограничивает пользователей в типах файлов. Важно лишь соблюдать их размер - не больше 25 Мб. При этом через терминал вы можете добавить файл весом до 100 Мб.

Уведомления о комментариях

https://support.google.com/youtube/answer/3523594?hl=ru ➔

Чтобы включить уведомления об ответах на комментарии в YouTube, нужно:

  1. Открыть сайт youtube.com и войти в свой аккаунт.
  2. Нажать на значок канала в правом верхнем углу страницы, выбрать «Настройки».
  3. В меню слева выбрать «Уведомления».
  4. В разделе «Общие» нажать «Ваши настройки».
  5. Выбрать, какие уведомления о действиях на своём канале и комментариях (в том числе об отметках «Нравится» к ним и ответах на них) хочется получать.

Для настройки уведомлений по электронной почте:

  1. Открыть раздел «Настройки уведомлений».
  2. В разделе «Уведомления по электронной почте» установить переключатель «Уведомления о последних действиях на моём канале и новости сервиса (кроме тех, на которые отменена подписка)» в положение «включено».

Также уведомления об ответах на комментарии можно включить в мобильном приложении YouTube, для этого нужно войти в него, выбрать свой логотип в правом верхнем углу, нажать «Настройки», затем «Уведомления» и включить функции «Активность в моих комментариях» и «Ответы на мои комментарии».
