или, если хочется иметь страничку на GitHub Pages, как у меня, то это будет ПУБЛИЧНЫЙ Репозиторий.

GitHub Pages - это сервис от GitHub, который позволяет разместить сайт в интернете, позволив заходить на него другим людям. Другими словами - это хостинг.

Сайт на GitHub Pages создается из репозитория, который должен быть публичным (то есть все ресурсы проекта будут видны другим пользователям) и иметь специальное имя. Такой репозиторий может быть всего один на одном аккаунте GitHub.
